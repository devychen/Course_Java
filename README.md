对于字符串 "happy"，其 .length() 方法返回的长度是 5。  
这是因为 String 的 length() 方法返回的是字符串中字符的数量。
在 Java 中，String.length() 计算的是字符数而不是字节数，因此它会按字符计数，包括空格、标点符号等。
